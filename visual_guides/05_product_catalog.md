# Product Catalog Visual Guide

## Overview
The product catalog system displays products from the store backend API, allowing users to browse, filter, and view product details.

## API Endpoints Used
- `GET /api/products/category/:categoryId` - Products by category
- `GET /api/products/search` - Search products
- `GET /api/products/:productId` - Single product details
- `GET /api/products/get-best-of-products` - Best products by period
- `GET /api/products/liked-products` - User's liked products
- `GET /api/deals/get-deals` - Current deals
