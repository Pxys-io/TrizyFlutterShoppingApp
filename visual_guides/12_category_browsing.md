# Category Browsing Visual Guide

## Overview
The category browsing system allows users to navigate through product categories and subcategories using the store backend API.

## API Endpoints Used
- `GET /api/categories/get-root-categories` - Get main categories
- `GET /api/categories/get-child-categories/:parentId` - Get subcategories
