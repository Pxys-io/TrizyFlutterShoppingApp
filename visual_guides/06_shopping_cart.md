# Shopping Cart Visual Guide

## Overview
The shopping cart system allows users to add, remove, and manage products they wish to purchase using the store backend API.

## API Endpoints Used
- `POST /api/carts/add-item` - Add item to cart
- `GET /api/carts/get-cart` - Get user's cart
- `GET /api/carts/get-cart-items` - Get cart item IDs
- `POST /api/carts/add-item-on-feed` - Add single item to cart
- `PATCH /api/carts/decrement-quantity` - Decrease item quantity
- `DELETE /api/carts/delete-item/:productId` - Remove item from cart
