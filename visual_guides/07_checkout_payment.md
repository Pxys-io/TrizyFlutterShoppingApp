# Checkout & Payment Visual Guide

## Overview
The checkout and payment system handles the order placement process and payment processing using the store backend API and Stripe.

## API Endpoints Used
- `POST /api/payments/create-payment-intent` - Create payment intent
- `POST /api/orders` - Create order (COD)
- `POST /api/carts/add-item` - Update cart after purchase
- `GET /api/address/get-all-addresses` - Get user addresses
- `GET /api/address/get-default-address` - Get default address
